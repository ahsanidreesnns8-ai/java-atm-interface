# 🏧 Java ATM Interface

> A console-based ATM Interface developed in Java using Object-Oriented Programming (OOP). This project simulates essential banking operations such as depositing money, withdrawing funds, checking account balance, and validating user input through a clean, menu-driven interface.

# 📖 Overview

The **Java ATM Interface** is a beginner-friendly console application built to demonstrate the practical implementation of **Object-Oriented Programming (OOP)** concepts in Java.

The application allows users to perform common ATM operations in a secure and interactive menu-driven environment while ensuring proper validation of user inputs.

This project was completed as **Project 3** during the **DecodeLabs Java Industrial Training Program**.

---

# ✨ Features

- 💰 Deposit Money
- 💸 Withdraw Money
- 💳 Check Account Balance
- 🔄 Interactive Menu-Driven Interface
- ✅ Input Validation
- 🔒 Encapsulation using Private Variables
- 🏗️ Composition-based OOP Design

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/your-username/java-atm-interface.git
```

## Navigate to Project

```bash
cd java-atm-interface
```

## Compile

```bash
javac DecodeLabs_Java_P3.java
```

## Run

```bash
java DecodeLabs_Java_P3
```
