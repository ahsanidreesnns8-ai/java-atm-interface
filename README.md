# 🏧 Java ATM Interface

A console-based ATM Interface developed in Java using Object-Oriented Programming (OOP). This project simulates basic banking operations such as depositing money, withdrawing funds, checking account balance, and validating user input through a menu-driven interface.

---

## 📌 Features

- 💰 Deposit money
- 💸 Withdraw money
- 💳 Check account balance
- 🔄 Menu-driven interface
- ✅ Input validation for menu and transaction amounts
- 🔒 Encapsulation using private data members
- 🏗️ Composition-based OOP design (`ATM` has a `BankAccount`)

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Java Scanner Class
- IntelliJ IDEA / VS Code

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/java-atm-interface.git
```

2. Open the project in your preferred Java IDE.

3. Compile and run:

```bash
javac DecodeLabs_Java_P3.java
java DecodeLabs_Java_P3
```
